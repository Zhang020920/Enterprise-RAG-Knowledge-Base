# 增强版企业制度语料说明

本目录用于放置更接近真实企业场景的复杂制度文档，供 RAG 检索、重排、引用来源展示和评估实验使用。

设计目标：

- 文档长度更接近真实企业制度，不再是几条简单规则。
- 每份制度包含适用范围、条款编号、审批路径、例外情况、表格标准和冲突处理。
- 同一主题在不同文档中存在边界和交叉，便于测试混合检索、重排、反思重试和来源引用。
- 保留原始 `data/raw/` 下的简单文档，用于对比“简单语料”和“复杂语料”的效果差异。

建议重建索引后测试：

- 北京出差住宿标准是多少？
- 超过住宿标准可以报销吗？
- 病假需要什么证明？工资怎么发？
- 连续请病假超过 10 天怎么办？
- 新员工可以领哪些 IT 设备？
- 采购电脑需要走什么审批？
- 外包人员可以申请 VPN 吗？
- 离职当天账号什么时候关闭？
- 客户招待费超过 3000 元怎么审批？
- 员工绩效申诉需要几天内提交？
